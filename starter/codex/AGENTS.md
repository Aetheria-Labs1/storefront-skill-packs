# Lexsis AI — Storefront Starter Pack

Foundation skills for building Shopify landing pages with AI. Works with any vertical or traffic source.

## MCP Server

Connect the Lexsis AI MCP server for full tool access:

```json
{
  "mcpServers": {
    "lexsis-ai": {
      "type": "http",
      "url": "https://mcp.trylexsis.com/mcp",
      "headers": { "Authorization": "Bearer <your-api-key>" }
    }
  }
}
```
